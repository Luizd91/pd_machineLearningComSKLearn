# pd_machineLearningComSKLearn
Entrega do PD da matéria de Machine Learning com sk-learn

Este PD foi realizado com base em um dataset sobre saúde de norte-americanos em relação a diabetes
Foi criado de modelo simples com perceptron, evoluindo para árvores de decisão e terminando em Random Forest
O dataset usado está anexado para quem quiser utilizar ficar a vontade
